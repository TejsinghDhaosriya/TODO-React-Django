# Movie-Booking

### Technologies

Frontend :- Reactjs ,Redux

### Deployment :-

      Frontend on Netlify

### Setup the Project locally :- <br/>

**_Frontend_**:-

1. Move to frontend folder
   cd frontend
2. Installing the dependencies
   yarn install
3. Starting the frontend
   yarn start
   will serve project at port 3000
